HTTP ステータスコード一覧

1xx〜5xx の HTTP ステータスコードの名称と意味の早見表。クラスへジャンプ、または Ctrl / ⌘ + F で検索できます。

1xx 情報 2xx 成功 3xx リダイレクト 4xx クライアントエラー 5xx サーバーエラー

1xx — 情報レスポンス

リクエストを受け取り、処理を継続していることを示します。

コード名称意味
100Continueリクエストの先頭部分を受理。クライアントは本文の送信を続けてよい。
101Switching Protocolsクライアントの要求に従いプロトコルを切り替える（例: WebSocket へ）。
102Processingリクエストは受理したが処理は未完了（WebDAV）。
103Early Hints最終応答の前に Link プリロード等のヘッダーを先送りする。

2xx — 成功

リクエストが正常に受理・処理されたことを示します。

コード名称意味
200OK標準的な成功応答。意味は HTTP メソッドに依存する（GET なら本文を返す等）。
201Createdリクエストが成功し、新しいリソースが作成された（POST/PUT の後など）。
202Accepted処理のために受理されたが、処理はまだ完了していない（非同期）。
203Non-Authoritative Information返すメタ情報が、オリジンではなく複製や変換プロキシ由来である。
204No Content成功したが、本文として返す内容はない。
205Reset Content成功後、フォーム等の表示をリセットするようクライアントに指示する。
206Partial ContentRange ヘッダーに応じてリソースの一部のみを返す（レジューム DL 等）。
207Multi-Status複数の操作それぞれに対する独立したステータスをまとめて返す（WebDAV）。
208Already ReportedWebDAV バインディングの要素が既に列挙済みで、繰り返さない。
226IM Used現在のリソースにインスタンス操作を適用した結果を返す。

3xx — リダイレクト

リクエスト完了には追加の操作が必要で、リソースが別の場所にあります。

コード名称意味
300Multiple Choices複数の応答候補があり、クライアントまたは利用者が選ぶ。
301Moved Permanentlyリソースが恒久的に新 URL へ移動。検索評価は新 URL に引き継がれる。
302Found一時的に別 URL にある。元の URL が正規のまま扱われる。
303See Other応答は別 URL にあり、GET で取得すべきことを示す。
304Not Modifiedキャッシュが有効なので再利用可能。条件付きリクエストで帯域を節約。
307Temporary Redirect302 と同様だが、リダイレクト時に HTTP メソッド（POST 等）を変えない。
308Permanent Redirect301 と同様でメソッドを変えない。Cloudflare Pages は末尾整形にこれを使う。

4xx — クライアントエラー

リクエストに問題があり、サーバーが処理できない場合です。

コード名称意味
400Bad Request構文の誤りや無効なパラメータのため、リクエストを処理できない。
401Unauthorized認証が必要だが、認証に失敗した、または未提供である。
402Payment Required将来のための予約。API が課金上の問題を示すのに使うことがある。
403Forbiddenリクエストは理解したが、権限不足で許可しない。
404Not Found要求されたリソースが見つからない。URL の誤りや削除が原因。
405Method Not Allowedそのリソースで許可されない HTTP メソッド（読取専用への POST 等）。
406Not Acceptableクライアントの Accept に合致する表現がない。
407Proxy Authentication Requiredまずプロキシで認証する必要がある。
408Request Timeoutクライアントからのリクエスト送信を待ってタイムアウトした。
409Conflictリソースの現在の状態と競合する（編集の衝突など）。
410Goneリソースは恒久的に消失し、転送先も不明。
411Length RequiredContent-Length ヘッダーが必要だが欠落している。
412Precondition Failedヘッダーの前提条件（If-Match 等）が満たされない。
413Content Too Largeリクエスト本文がサーバーの処理可能な上限を超えている。
414URI Too Long要求 URI がサーバーの解釈できる長さを超えている。
415Unsupported Media Type本文の形式（Content-Type）がリソースで未対応。
416Range Not Satisfiable要求された Range がリソースのサイズ外で提供できない。
417Expectation FailedExpect ヘッダーの期待を満たせない。
418I'm a teapotエイプリルフールのジョークコード（RFC 2324）。遊びで返されることがある。
421Misdirected Request応答を生成できないサーバーへリクエストが送られた。
422Unprocessable Content構文は正しいが意味的な誤りがある（バリデーション API で多用）。
423Lockedアクセス対象のリソースがロックされている（WebDAV）。
424Failed Dependency依存する別リクエストが失敗したため失敗した（WebDAV）。
425Too Early再送される恐れのあるリクエストの処理を拒む。
426Upgrade RequiredUpgrade ヘッダーで示す別プロトコルへの切替が必要。
428Precondition Required条件付きリクエストを要求（更新の取りこぼしを防ぐ）。
429Too Many Requests一定時間に送りすぎ（レート制限）。
431Request Header Fields Too Largeヘッダーフィールドが大きすぎて処理できない。
451Unavailable For Legal Reasons法的要請（検閲・削除要求など）で利用できない。

5xx — サーバーエラー

正当なリクエストに対し、サーバー側が処理に失敗した場合です。

コード名称意味
500Internal Server Error汎用エラー。サーバー内部で予期しない状態が発生し処理できなかった。
501Not Implementedリクエストの実現に必要な機能をサーバーが備えていない。
502Bad Gatewayゲートウェイ／プロキシが上流サーバーから無効な応答を受け取った。
503Service Unavailable過負荷やメンテナンスで一時的に利用不可。時間をおいて再試行。
504Gateway Timeoutゲートウェイ／プロキシが上流から時間内に応答を得られなかった。
505HTTP Version Not Supportedリクエストの HTTP バージョンをサーバーが未対応。
506Variant Also Negotiatesサーバー側のコンテンツネゴシエーション設定エラー。
507Insufficient Storage処理に必要な表現を保存できない（WebDAV）。
508Loop Detected処理中に無限ループを検出した（WebDAV）。
510Not Extended要求の実現にはさらなる拡張が必要。
511Network Authentication Requiredネットワーク利用に認証が必要（キャプティブポータルで多い）。

よくある質問

404 の意味は？ サーバーが要求されたリソースを見つけられない状態です。URL の誤りや、ページの移動・削除が原因です。

301 と 302 の違いは？ 301 は恒久的リダイレクトで検索評価を新 URL に引き継ぎ、302 は一時的で元の URL が正規のまま扱われます。

502 Bad Gateway とは？ ゲートウェイ／プロキシとして動くサーバーが、頼った上流サーバーから無効な応答を受け取った状態です。
