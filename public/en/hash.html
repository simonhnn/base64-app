About hashing

A hash function maps data of any length to a fixed-length value in one direction. It is used for file integrity checks and data fingerprints. This tool uses the browser's built-in Web Crypto API to compute SHA-256, SHA-1, SHA-384 and SHA-512 (MD5 is not supported). Your input is never sent to a server.
