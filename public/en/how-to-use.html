How to use

Convert between text and Base64 instantly.

Basic steps

  1. Type a string into either the top field (Base64) or the bottom field (plain text).
  2. Press the "Convert" button in the middle.
  3. The result appears in the other field based on which side you edited.
  4. Use the copy icon at the bottom-right of each field to copy the result. Press "Clear" to reset.

Notes

  • The input limit is 10,000 characters.
  • If the format is invalid when decoding, an error is shown.
  • Conversion runs entirely inside your browser.
