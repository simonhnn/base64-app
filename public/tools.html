オンライン変換ツール一覧

すべてブラウザ内で完結。インストール・登録は不要です。

  • Base64 エンコード・デコード テキストと Base64 を相互変換します。
  • URL エンコード・デコード パーセントエンコーディングを相互変換します。
  • JSON 整形・圧縮 JSON を見やすく整形、または圧縮・文法チェックします。
  • ハッシュ生成（SHA） SHA-256/1/384/512 のハッシュ値を計算します。
  • 年齢早見表 西暦・和暦から満年齢がわかる早見表（干支つき）。
  • 和暦西暦変換表 明治〜令和と西暦の対応表。改元年の旧元号・干支つき。
  • HTTP ステータスコード一覧 1xx〜5xx の意味を早見表で確認（200・301・404・500 ほか）。
